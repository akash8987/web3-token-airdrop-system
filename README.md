# Web3 Token Airdrop System

This is a beginner-friendly Web3 project.  
It helps users connect their wallet and claim free test tokens from a smart contract.

This project uses:
- Solidity for smart contract
- JavaScript for frontend
- Ethers.js for blockchain connection

## What This Project Does

- User opens the website
- User connects MetaMask wallet
- User clicks "Claim Token"
- Smart contract sends tokens to user wallet

## Folder Structure

- contracts/ = Smart contract files
- frontend/ = Website files
- scripts/ = Deploy scripts
- config/ = Network settings

## Requirements

- Node.js
- MetaMask Wallet
- Test network like Sepolia or Goerli
